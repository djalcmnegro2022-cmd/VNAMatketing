# 🎧 VNA Marting IA 🇦🇴
**Desenvolvido por Dj M Negro | VNAStudio**

O **VNA Marting IA** é uma ferramenta exclusiva criada para revolucionar o marketing de produtores musicais em Angola. Este projeto integra inteligência artificial simulada para gerar ganchos virais e descrições para Beats de Trap, Kuduro, Amapiano e Afrobeat.

## 🚀 Funcionalidades
* **Gerador de Marketing IA:** Cria frases de impacto para TikTok e Instagram.
* **Player Integrado:** Ouça as playlists oficiais do VNA diretamente no app.
* **Calculadora de Estúdio:** Gestão de lucros e custos de produção.
* **Sistema de Cópia Rápida:** Facilita a postagem de conteúdos nas redes sociais.
